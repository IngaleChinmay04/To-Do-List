# To-Do List
A simple task manager built with HTML, CSS, and JavaScript using Dragula and Vanilla JS.

![screenshot](image.png)

## Features
- Add new tasks
- Move tasks between columns: To Do, Doing, Done, and Trash
- Clear tasks in the Trash column
